# ruixu2.github.io

个人导航页，提供以下功能：

## 功能特性

### 📦 GitHub Pages 项目导航
- 自动加载并展示 ruixu2 用户下所有启用了 GitHub Pages 的仓库
- 点击即可跳转到对应的子站点

### 🔍 多搜索引擎入口
提供三个独立的搜索框，默认分别为：
- **Bing 搜索** - 微软搜索引擎
- **Google 搜索** - 谷歌搜索引擎  
- **百度搜索** - 百度搜索引擎

### ⚙️ 自定义搜索引擎
每个搜索框都支持：
- 下拉选择切换不同的搜索引擎（Bing、Google、百度、DuckDuckGo、Yahoo）
- 选择"自定义..."可输入任意搜索引擎 URL
- 支持使用 `{query}` 作为查询占位符

## 使用方式

访问 https://ruixu2.github.io 即可使用导航页。

## 技术栈

- 纯 HTML/CSS/JavaScript 实现
- 响应式设计，支持移动端
- 通过 GitHub API 动态获取仓库列表
